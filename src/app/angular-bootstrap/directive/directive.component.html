We have 3 types of directives in angular

  1. Component Directive
  2. Attributive Directive
  3. Structural Directive

1: Component Directive

Component directive are most common used directives
when you create a component so @Component decorator has the metadata

Structural directives are type of directives that actually
allows you to manipulate the DOM(document object model)based on conditions

but we can make our custom structural-directive as well Click here to see the example

Attribute directives are type of directives that actually change the behavior
